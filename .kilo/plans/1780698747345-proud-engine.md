## Plugin Structure Compatibility

### Existing Patterns to Follow
- All functions use global `$wpdb` with `$wpdb->prefix`
- SQL queries use `$wpdb->prepare()` for parameterized queries
- AJAX handlers: `check_ajax_referer()`, `current_user_can('manage_gmcq')`, `wp_send_json_*`
- Cache invalidation uses `gmcq_clear_dashboard_cache('quiz')`
- Transaction pattern used in `gmcq_set_quiz_questions()` (START TRANSACTION/COMMIT/ROLLBACK)

### No Breaking Changes Guaranteed
- Adding LEFT JOIN does not break existing callers
- LEFT JOIN ensures backward compatibility with quizzes without category
- AJAX enhancement is additive (new params optional)
- All changes follow existing coding style and WordPress conventions
- No function signature changes
